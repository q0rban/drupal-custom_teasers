DESCRIPTION
------------
Creates a teaser content type, allowing users to create a custom teaser with 
images that link to a specific piece of content.  Some features:

 * Multiple teasers linking to one piece of content
 * Create views that alter the text based on which view is being 
   used.
 * Themes may override which imagecache preset is being used, or which fields
   display, or the length of the text of a particular field.

INSTALLATION
-----------
Copy the module to your modules directory and enable.  Then navigate to 
admin/settings/custom_teasers and import the cck imagefield into the teaser
content type.
